# altgo 文档

欢迎使用 **altgo**：基于 **Tauri + React + Rust** 的 Linux 语音转文字桌面应用（目前主要在 **Ubuntu 20.04** 上验证）。

按住触发键（默认**右 Alt**）说话，松开后在本地用 **whisper.cpp** 转写，并可选用 **OpenAI 兼容 API** 做 LLM 润色。转写成功后会**尝试写入系统剪贴板**，同时用**悬浮窗**展示结果，方便核对或在剪贴板失败时手动复制。

## 核心特性一览

| 特性 | 说明 |
|------|------|
| **长按即录** | 按住右 Alt 说话，松开自动转写 |
| **连续模式** | 双击进入连续录音，适合长篇发言 |
| **本地 ASR** | whisper.cpp 离线转写，无需 API 密钥 |
| **云端 ASR** | 支持 OpenAI Whisper API |
| **LLM 润色** | 四档强度，支持多种 Provider |
| **剪贴板 + 悬浮窗** | 自动写入剪贴板，悬浮窗可二次复制 |
| **平台支持** | Linux（X11 / Wayland） |
