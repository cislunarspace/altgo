# altgo 文档

欢迎使用 **altgo**：基于 **Tauri + React + Rust** 的跨平台语音转文字桌面应用，默认以 **Linux** 为第一目标平台（目前主要在 **Ubuntu 20.04** 上验证）。

按住触发键（默认**右 Alt**）说话，松开后在本地用 **whisper.cpp** 转写，并可选用 **OpenAI 兼容 API** 做 LLM 润色。转写成功后会**尝试写入系统剪贴板**，同时用**悬浮窗**展示结果，方便核对或在剪贴板失败时手动复制。
