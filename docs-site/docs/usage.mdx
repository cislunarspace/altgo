# 使用说明

## 录音模式

### 长按模式

最常用的模式。按住 **右 Alt** 键说话，松开自动停止录音并处理。

```
按下右 Alt ──→ 开始录音
   │
   │  说话中...
   │
松开右 Alt ──→ 停止录音 → 转写 → (润色) → 写入剪贴板
```

### 连续模式

适合较长的发言。**双击右 Alt** 进入连续录音，再次 **单击** 停止。

```
双击右 Alt ──→ 进入连续录音
   │
   │  持续录音中...
   │
单击右 Alt ──→ 停止录音 → 转写 → (润色) → 写入剪贴板
```

:::tip 提示
连续模式下你可以暂停思考，不需要一直按住按键。
:::

## 处理流程

录音结束后，altgo 会依次执行：

1. **转写** — 将音频发送到 Whisper（本地或云端）
2. **润色**（可选）— 通过 LLM 优化文本
3. **输出** — 写入系统剪贴板 + 桌面通知

整个流程自动完成，你只需松开按键，等待结果出现在剪贴板中。

## 润色级别

| 级别 | 适用场景 |
|------|---------|
| `none` | 实时聊天、命令输入，需要原话 |
| `light` | 修正标点和错别字，保持原话风格 |
| `medium` | 日常写作，修正语法和语序 |
| `heavy` | 正式文档、邮件，结构化重写 |

## 输出方式

altgo 完成转写后：

- **自动写入剪贴板** — 在任何应用中直接 `Ctrl+V` / `Cmd+V` 粘贴
- **桌面通知** — 弹出通知显示处理结果（可在配置中关闭）
- **光标注入**（仅 Windows）— 尝试自动将文本输入到当前光标位置

## 桌面 GUI 版本

除了 CLI 版本，altgo 还提供 Tauri 桌面应用（开发中），提供：

- 系统托盘图标
- 可视化设置界面
- 录音状态浮窗
- 转写历史记录

启动 GUI 版本：

```bash
cd src-tauri && cargo tauri dev
```

## 命令行参数

```bash
altgo                        # 前台运行，使用默认配置
altgo --config /path/to/altgo.toml  # 指定配置文件
altgo --version              # 查看版本
altgo --help                 # 查看帮助
```
